 Neste projeto é abordado um script que será utilizado em diversos problemas de machine  learning. Nele será visto extração dos dados, limpeza dos dados, analise exploratória dos dados, gráficos de linha, de violinos, pairplots e correlação, criação de novas variáveis a partir das pré-estabelecidas, cross-validation, escala e pré-processamento dos dados, varios modelos de machine learning, previsão dos dados de treino e dados de teste, otimização de hiperparametros e métricas.

 O intuito do projeto é mostrar que com um mesmo script(em estrutura, já que algumas partes, como criação de novas variáveis e modelos de machine learning são diferentes para os diferentes problemas propostos) podemos resolver diversos problemas na area de ciência de dados.

 Foi utilizado a linguagem de programação python e as bibliotecas mais utilizadas em todos os trabalhos foram: pandas, numpy, matplotlib e principalmente o sklearn.

 Vale ressaltar que nos projetos extras não é tratada a análise exploratória dos dados. Além disto, os datasets foram retirados do kaggle.
